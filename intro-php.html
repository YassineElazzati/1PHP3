01
L'ossature

Un fichier PHP, c'est du HTML avec des îlots de code.

Un fichier .php ressemble à un fichier HTML classique. La différence : on peut y glisser des blocs de code délimités par <?php et ?>. Le serveur les exécute, et remplace chaque bloc par son résultat avant d'envoyer la page au navigateur.

Lisez chaque ligne avec son explication à droite. C'est la structure que vous croiserez dans 90 % des fichiers PHP au début.

01 <!DOCTYPE html>
Déclare au navigateur que c'est un fichier HTML5. Toujours en première ligne, obligatoire.
02 <html lang="fr">
Ouvre la racine de la page. lang="fr" dit au navigateur et à Google que la page est en français.
03 <head>
Début de la zone d'en-tête. Ce qui s'y trouve ne s'affiche pas dans la page, c'est de la méta-info.
04 <meta charset="UTF-8">
Indique l'encodage des caractères. UTF-8 permet d'afficher les accents (é, è, à...) et les emojis correctement.
05 <title>Ma première page PHP</title>
Définit le titre affiché dans l'onglet du navigateur (et dans les résultats Google).
06 </head>
Ferme la zone d'en-tête.
07 <body>
Début du contenu visible dans la fenêtre du navigateur.
08 <h1>Bienvenue</h1>
Un titre HTML statique. PHP ne le touche pas, il arrive tel quel au navigateur.
09
Ligne vide pour aérer le code. N'a aucun effet sur le rendu.
10 <?php
Début du bloc PHP. Tout ce qui suit jusqu'à ?> sera exécuté sur le serveur. Le navigateur ne verra jamais ce code, seulement son résultat.
11 // Code PHP ici - sera exécuté côté serveur
Un commentaire. Le // dit à PHP : « ignore cette ligne ». Sert juste à expliquer le code au futur lecteur (toi, dans 3 mois).
12 echo "<p>Cette ligne est générée par PHP</p>";
echo = « affiche-moi ça ». Imprime la chaîne entre guillemets dans la page HTML. Le ; à la fin termine l'instruction — obligatoire.
13 ?>
Fin du bloc PHP. En dessous, on est de retour en HTML pur.
14
15 <p>Cette ligne est du HTML statique</p>
Du HTML normal, en dehors du bloc PHP. Identique pour tous les visiteurs.
16 </body>
Fin de la zone visible.
17 </html>
Fin du document.

Les 5 symboles à retenir

<?php
Ouvre un bloc PHP. Tout ce qui suit est exécuté côté serveur.
?>
Ferme le bloc PHP. Optionnel en toute fin de fichier.
echo
Affiche du texte ou du HTML dans la page reçue par le visiteur.
//
Commentaire sur une seule ligne. Ignoré par PHP.
;
Termine chaque instruction PHP. L'oublier = erreur de syntaxe.
◆
PHP s'insère dans le HTML, pas l'inverse

Le HTML est la base. PHP est un invité qui s'installe par endroits, fait son travail, et disparaît avant que la page ne parte vers le navigateur. À l'arrivée, le visiteur ne voit que du HTML — toujours.

02
Les étiquettes

Stocker une valeur, c'est poser une étiquette sur une boîte.

Une variable, c'est un nom (l'étiquette) qui pointe vers une valeur (le contenu de la boîte). En PHP, toute variable commence par le symbole $. Ce symbole dit au moteur PHP : « attention, c'est une variable, pas un mot normal ».

01 <?php
On ouvre le bloc PHP.
02 $nom = "Marie";
Crée une variable $nom et y range la chaîne "Marie". C'est un String (texte). Les guillemets sont obligatoires autour du texte.
03 $age = 25;
Crée $age avec la valeur 25. C'est un Integer (nombre entier). Pas de guillemets autour des nombres.
04 $taille = 1.68;
Crée $taille avec 1.68. C'est un Float (nombre décimal). Attention : on utilise un point, pas une virgule.
05 $estEtudiant = true;
Crée $estEtudiant à true. C'est un Boolean (vrai ou faux). Seules deux valeurs possibles : true ou false.
06
07 echo "Nom : " . $nom;
Concaténation. Le . colle deux textes ensemble. Affiche : Nom : Marie.
08 echo "<br>";
Affiche une balise HTML <br> = saut de ligne dans le navigateur.
09 echo "Age : $age ans";
Interpolation. Dans des guillemets doubles, PHP remplace $age par sa valeur. Affiche : Age : 25 ans. Plus court que la concaténation.
10 ?>
On ferme le bloc PHP.

Les 6 types de données

Chaque variable contient une seule donnée d'un type précis. PHP détecte automatiquement le type selon ce que vous mettez dedans (c'est ce qu'on appelle le typage faible) :

TypeDescriptionExemple
String Chaîne de caractères (texte) $texte = "Bonjour";
Integer Nombre entier $nombre = 42;
Float Nombre à virgule (décimal) $prix = 19.99;
Boolean Vrai ou faux $actif = true;
Array Tableau (plusieurs valeurs dans une boîte) $liste = [1, 2, 3];
NULL Absence de valeur (la boîte est vide) $vide = null;
▲
Trois règles qui sauvent des heures de debug

Sensibilité à la casse. $nom et $Nom sont deux variables différentes. PHP fait la différence entre majuscule et minuscule.

Convention camelCase. Pour les noms à plusieurs mots, on colle les mots avec une majuscule au milieu : $monAge, $prenomClient, $totalCommandes.

Toujours commencer par $. Sans le dollar, PHP cherche une constante ou une fonction et lève une erreur.

▼ E-commerce · Cdiscount

Vos variables, sur un vrai site

Quand vous consultez votre profil sur Cdiscount, le serveur PHP récupère vos infos en variables : $nomClient = "Dupont", $emailClient = "dupont@example.com", $totalCommandes = 1250.50. Ces variables sont ensuite injectées dans le HTML pour générer votre page profil, personnalisée. Les variables, c'est ça : la mémoire temporaire de la page pendant qu'elle se construit.

03
Les outils de calcul

Calculer, comparer, décider : les opérateurs.

Un opérateur, c'est un symbole qui prend une ou deux valeurs et retourne un résultat. On les classe en deux familles : les arithmétiques (qui font des maths) et les comparaisons (qui retournent vrai ou faux).

Opérateurs arithmétiques

01 $a = 10;
Range 10 dans $a.
02 $b = 3;
Range 3 dans $b.
03 echo $a + $b;
Addition. Affiche 13.
04 echo $a - $b;
Soustraction. Affiche 7.
05 echo $a * $b;
Multiplication. Affiche 30.
06 echo $a / $b;
Division. Affiche 3.333... (résultat décimal).
07 echo $a % $b;
Modulo (reste de la division). 10 ÷ 3 = 3, reste 1. Affiche 1. Très utile pour tester la parité : $n % 2 === 0 = vrai si pair.
08 echo $a ** $b;
Puissance. 10 puissance 3 = 1000.

Opérateurs de comparaison : == vs ===

C'est le piège n°1 du débutant PHP. Comparer deux valeurs, c'est demander : « ces deux choses sont-elles égales ? ». Mais PHP propose deux niveaux d'égalité.

Expression
Résultat
Pourquoi
5 == "5"
true
== compare juste la valeur. PHP convertit silencieusement "5" en nombre et trouve 5 = 5.
5 === "5"
false
=== compare valeur ET type. 5 est un Integer, "5" est un String → types différents.
5 != "5"
false
L'inverse de ==. Les valeurs sont égales (après conversion), donc « différent » est faux.
5 !== "5"
true
L'inverse de ===. Les types sont différents, donc « strictement différent » est vrai.
◆
La règle de survie : toujours ===

Utilisez === par défaut. == autorise des conversions silencieuses qui causent des bugs vicieux. Avec ===, vous comparez ce que vous voyez, sans mauvaise surprise.

04
Les aiguillages

Faire des choix : si... alors... sinon...

Un programme sans condition fait toujours la même chose. Les conditions permettent à votre code de réagir différemment selon les valeurs qu'il rencontre. C'est ce qui transforme un script linéaire en un vrai programme intelligent.

01 $note = 14;
On range la note 14 dans $note. C'est la valeur qu'on va tester.
02
03 if ($note >= 16) {
Premier test. Si $note ≥ 16, on entre dans ce bloc. Les { } délimitent le code à exécuter.
04 echo "Très bien !";
Ce qui s'affiche uniquement si la condition de la ligne 03 est vraie.
05 } elseif ($note >= 12) {
Sinon si la note est ≥ 12. PHP n'arrive ici que si la condition précédente était fausse.
06 echo "Bien";
S'affiche pour 12 ≤ note < 16.
07 } elseif ($note >= 10) {
On peut chaîner autant de elseif qu'on veut.
08 echo "Passable";
S'affiche pour 10 ≤ note < 12.
09 } else {
Sinon (le filet de sécurité). Exécuté si aucune condition précédente n'est vraie.
10 echo "Insuffisant";
S'affiche pour toute note < 10.
11 }
Ferme la chaîne if/elseif/else.

L'aiguillage en visuel

Avec $note = 14, PHP teste chaque condition dans l'ordre. La première vraie gagne, les suivantes sont ignorées :

▼ Réservation · Airbnb

Quand un utilisateur réserve un logement

Avant de valider, le serveur PHP enchaîne plusieurs conditions : if ($disponible) vérifie que les dates sont libres, if ($solde >= $prix) contrôle que le compte peut payer, if ($identiteVerifiee) s'assure de l'identité. Selon les résultats, le script génère soit une confirmation, soit un message d'erreur précis. Toute interface web qui doit décider quelque chose repose sur ce mécanisme.

05
Les répétitions

Répéter sans copier-coller : les boucles.

Une boucle dit à PHP : « refais ça N fois ». Sans elle, il faudrait écrire 50 fois la même ligne pour afficher 50 articles. Avec elle, 3 lignes suffisent — quelle que soit la taille du tableau.

La boucle for — quand on sait combien de fois

01 for ($i = 1; $i <= 10; $i++) {
Trois infos séparées par des ; :
• $i = 1 → départ, on commence à 1.
• $i <= 10 → condition, on continue tant que vrai.
• $i++ → incrément, on ajoute 1 à chaque tour.
02 echo "Nombre : $i <br>";
Le corps de la boucle. Exécuté une fois par tour. $i vaut 1, puis 2, puis 3... jusqu'à 10.
03 }
Ferme la boucle. Total : 10 affichages avec 3 lignes de code.

La boucle foreach — parcourir un tableau

Quand on a déjà une liste (un tableau), on n'a pas besoin de compteur. foreach se charge tout seul de visiter chaque élément.

01 $fruits = ["Pomme", "Banane", "Orange"];
Crée un tableau de 3 strings. Les crochets [ ] définissent un array. Les valeurs sont séparées par des virgules.
02
03 foreach ($fruits as $fruit) {
Lit ça : « pour chaque élément de $fruits, range-le dans $fruit ». À chaque tour, $fruit prend la valeur suivante : "Pomme", puis "Banane", puis "Orange".
04 echo "J'aime les $fruit <br>";
Exécuté 3 fois (un par fruit). Affiche : J'aime les Pomme / J'aime les Banane / J'aime les Orange.
05 }
Ferme la boucle.

Variante avec index

Parfois on veut aussi la position de chaque élément (0, 1, 2...). On utilise la syntaxe as $cle => $valeur :

01 foreach ($fruits as $index => $fruit) {
Cette fois on récupère deux choses : la position dans $index (0, 1, 2...) et la valeur dans $fruit.
02 echo "$index : $fruit <br>";
Affiche : 0 : Pomme / 1 : Banane / 2 : Orange. Note : les index de tableau commencent toujours à 0, pas à 1.
03 }
Fin.
▶ Streaming · Netflix

« Ma liste » : la boucle en production

Quand vous ouvrez votre liste sur Netflix, le serveur récupère vos films favoris dans un tableau : $maListe = ["Stranger Things", "Breaking Bad", "The Crown"]. Une boucle foreach génère ensuite le HTML de chaque vignette. Si vous avez 50 films, le même code PHP s'exécute 50 fois, en adaptant les données à chaque tour. C'est pour ça qu'un site peut afficher des millions de contenus sans écrire des millions de lignes.
